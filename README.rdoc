## Backup/Restore locally

* `heroku pgbackups:capture` (create current backup, but it's automated anyway)
* `heroku pgbackups:url` (latest backup)
* Download from URL
* `pg_restore --verbose --clean --no-acl --no-owner -h localhost -U myuser -d mydb latest.dump`
