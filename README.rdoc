* `/admin` to flip categories

## Production

* Push latest json
* `ruby script/importer.rb`
* `/admin` to flip categories

## Backup/Restore locally

* `heroku pgbackups:capture` (create current backup, but it's automated anyway)
* `heroku pgbackups:url` (latest backup)
* Download from URL, put copy in tmp/backup.dump
* `bin/rake db:reset`
